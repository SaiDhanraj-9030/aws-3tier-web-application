# AWS 3-Tier Web Application Architecture

AWS 3-Tier Web Application Architecture using EC2, VPC, ALB, Auto Scaling, RDS, CloudWatch and SNS.

## Overview

Designed and deployed a highly available 3-Tier Web Application Architecture on AWS.

## Services Used

* Amazon VPC
* Public & Private Subnets
* Internet Gateway
* NAT Gateway
* Route Tables
* Security Groups
* Amazon EC2
* Application Load Balancer
* Auto Scaling Groups
* Amazon RDS
* Amazon Aurora
* MySQL

## Project Highlights

* Created secure network architecture using VPC.
* Configured public, private, and database subnets.
* Implemented Load Balancer and Auto Scaling.
* Deployed EC2 instances for application hosting.
* Configured RDS/Aurora for database services.
* Performed MySQL database operations.
* Implemented high availability and scalability.
